📚 템플릿 언어란?

 
        HTML에서 파이썬의 변수나 제어문 등을 사용할 수 있도록 도와주는 언어이다. 
        HTML만으로는 정적인 웹페이지만 만들 수 있는데, 템플릿 언어를 이용하면 동적으로 데이터를 
        표시하거나 반복, 조건 처리가 가능하다.

  • {{ 리스트명.0 }}

    • {% else %}
    • {% endif %} : 조건문 끝
  • 반복문
    • {% for item in 리스트 %}
    • {% endfor %} : 반복문 끝

📚 템플릿 주석

기본 레이아웃을 정의해두고, 하위 템플릿에서 공통 구조를 재사용할 수 있는 기능

            `{% extends "base.html" %}` 는 base.html을 가져다 쓰겠다는 뜻
→ 그 안에 있는 `{% block content %}` 구간에 자식 템플릿이 내용을 "채워 넣는" 구조
{% endblock %}
